# Sip & Play - Codédex Summer Hackathon 2024

This repository contains the source code for a custom website built for **Sip & Play**, a board game cafe located in Park Slope, Brooklyn. This project was developed as our team's submission for the **Codédex Summer Hackathon 2024** (Track 1: Board Game Cafe Website) held on July 11-12th, 2024.

The goal of this project is to create an engaging, interactive, and responsive web experience that incorporates 3D elements, an events calendar, and an approachable UI/UX for the cafe.

## Challenge

Create a brand new website using HTML, CSS, JavaScript, and Three.js or A-frame for **Sip & Play**, a board game shop serving boba tea and small bites in Park Slope, Brooklyn.

[Watch the video here to hear Sip & Play’s owner, Jonathan, talk about the details.](https://www.instagram.com/codedex.io/)

## Criteria

- Add an events calendar
- Ensure it’s an approachable UI/UX design
- Add a 3D menu with Three.js or A-Frame
- Mobile-friendly (responsive)
- SEO-friendly (meta info)
- Be as creative as you’d like!

## Technologies

- HTML, CSS, JavaScript, React, Next.js
- Three.js / React Three Fiber / @react-three/drei
- Tailwind CSS
- Vercel
